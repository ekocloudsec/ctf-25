Challenge 04 - CloudCopy Attack

Stealing NTDS Hashes via EBS Snapshots

üî• Advanced

üé≠ Scenario

EkoCloudSec has deployed a Windows Server 2022 Domain Controller for the ekocloudsec.local domain in AWS. The infrastructure team has created EBS snapshots for backup purposes, but the IAM permissions are misconfigured, allowing unauthorized users to access and share these snapshots.

Your mission is to exploit this misconfiguration using the CloudCopy technique to extract NTDS hashes from the Domain Controller's snapshot and compromise the service account credentials.

üéØ Objectives

  • Enumerate EBS snapshots in the target AWS account
  • Identify the Domain Controller snapshot containing NTDS database
  • Exploit IAM permissions to share the snapshot with your personal account
  • Extract NTDS.dit and SYSTEM registry hive from the snapshot
  • Use secretsdump.py to extract password hashes
  • Find the NT hash of the svc-flag service account
